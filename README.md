# Chick-fil-A Ordering System

A Python-based ordering system that simulates a restaurant ordering workflow.  
The program allows users to select menu items, calculate totals, and generate an order receipt.

This project demonstrates core programming concepts including input validation, control flow, and modular program design.

## Features

- Interactive command-line ordering system
- Menu selection and quantity input
- Real-time price calculation
- Order summary and receipt generation
- Input validation for user entries

---

## Technologies

- Python
- Control flow (loops and conditionals)
- Functions and modular program design

---

## System Workflow

User selects menu items
→
Program validates input
→
Prices are calculated
→
Receipt is generated
